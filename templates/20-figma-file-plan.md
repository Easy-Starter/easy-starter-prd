## Naming
- Frames: `<FLOW-ID> / <step> / <state>`
- Components: `<category>/<component>/<variant>`
- Variables: semantic tokens, not raw visual names

## Required States
Default, loading, empty, error, offline, disabled, permission denied, success, and destructive confirmation where relevant.

## Handoff Checklist
- [ ] RTL and LTR behavior documented
- [ ] Components use Auto Layout and variables
- [ ] Interactive states are represented
- [ ] Copy is real or clearly labeled placeholder
- [ ] Analytics event IDs appear in annotations
- [ ] Requirement and screen IDs are linked
- [ ] Motion and accessibility behavior are noted
