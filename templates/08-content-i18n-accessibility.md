## Accessibility Checklist
- [ ] Semantic headings
- [ ] Keyboard-only flow
- [ ] Visible focus state
- [ ] Color is not the only signal
- [ ] Form fields have labels
- [ ] Error messages explain fixes
- [ ] Images have alt text or are decorative
- [ ] Reduced motion is supported
